# Responsive Email Campaign System

A production-ready responsive HTML email template built using table-based layouts and inline CSS to ensure compatibility across major email clients such as Gmail, Outlook, and mobile devices.

---

## 🚀 Features

* Table-based layout for Outlook compatibility
* Inline CSS for consistent cross-client rendering
* Responsive design using media queries
* Preheader text to improve open rates
* Bulletproof call-to-action (CTA) buttons
* Optimized 600px layout standard
* Cross-platform support (Gmail, Outlook, Mobile)

---

## 🛠 Tech Stack

* HTML5
* CSS3 (Inline for email compatibility)

---

## 📂 Project Structure

* `index.html` – Main email template (production-ready with inline CSS)
* `styles.css` – Optional styling for local preview

---

## ▶️ Usage

1. Open `index.html` in your browser for preview
2. Customize content (text, images, links)
3. Use the inline CSS version for email campaigns

---

## ⚠️ Important Notes

* JavaScript is **not supported** in most email clients
* Inline CSS is required for proper rendering
* Table-based layout is used for Outlook compatibility
