# Email Campaign Template System

Responsive email template starter with inline CSS and optional JS helper.

## Files

- `index.html`: main email template preview (includes inline CSS for email client compatibility).
- `styles.css`: ancillary style guide for web preview and shared tokens.

## Run

1. Open `index.html` in your browser for local preview.
2. Edit the `<!-- CONTENT BLOCK -->` sections as needed.
3. Use the `Inline CSS` block from `index.html` for production email sends.

## Notes

- Emails generally do not support JS in inboxes, so `script.js` is for local CMS/tooling only.
- Use inline CSS for `.desktop` / `.mobile` styles and media queries for clients that support them.

## Core Features

- Designed and developed responsive HTML email templates for marketing campaigns.
- Implemented inline CSS to ensure compatibility across Gmail, Outlook, and mobile devices.
- Ensured cross-client compatibility and Outlook-safe table layout for reliable render in enterprise inboxes.
- Optimized layouts for different screen sizes ensuring high deliverability and readability.
- Created reusable templates for newsletters and promotional campaigns.
